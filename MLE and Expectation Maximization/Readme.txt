For executing the Matlab coding Assignment Problems,

1. Open the first.m program and run this file. Both the functions, colormle and colorem are called from this file.
2. First, enter the number of nodes.
3. Enter the values for the adjacency matrix row-wise.
4. Then, enter the size for the weight matrix and their values.
5. Then, enter the latent variable (This input is for the colorem problem).
6. The solution for colormle and colorem gets displayed.
